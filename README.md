# Korean Language Exam Practice Website

I developed a web application for users to practice Korean language exams with multiple question sets. Each set contains 20 reading and 20 listening questions. Before starting an exam, all necessary files are cached for a seamless experience. The admin panel allows administrators to add and manage multiple sets along with their corresponding questions.
